# Bilder / Bildwelt

Hier kommen die echten Fotos der Website hinein. **Solange eine Datei fehlt, zeigt die
Website automatisch den gestalteten Marken-Farbverlauf** — es entsteht kein „kaputtes Bild".
Sobald du eine Datei mit dem passenden Namen hier ablegst, erscheint das Foto sofort.

## Erwartete Dateien

| Datei | Wo | Format | Empfohlen |
|---|---|---|---|
| `hero.jpg` | Startseite, großes Bild oben rechts | JPG/WebP | Querformat ~4:3, ≥ 1200 px breit |
| `team-werkstatt.jpg` | Abschnitt „Unternehmen" | JPG/WebP | Hochformat ~4:5, ≥ 900 px hoch |
| `leistung-waermepumpe.jpg` | Leistungskarte Wärmepumpen | JPG/WebP | Querformat 16:9, ~800 px breit |
| `leistung-heizung.jpg` | Leistungskarte Heizsysteme | JPG/WebP | 16:9, ~800 px |
| `leistung-klima.jpg` | Leistungskarte Klimatechnik | JPG/WebP | 16:9, ~800 px |
| `leistung-bad.jpg` | Leistungskarte Sanitär & Bad | JPG/WebP | 16:9, ~800 px |
| `leistung-wartung.jpg` | Leistungskarte Wartung | JPG/WebP | 16:9, ~800 px |
| `leistung-notdienst.jpg` | Leistungskarte Notdienst | JPG/WebP | 16:9, ~800 px |
| `referenz-1.jpg` … `referenz-6.jpg` | Referenzen-Galerie | JPG/WebP | Querformat 4:3, ~800 px |

## Hinweise

- **Dateinamen exakt** so verwenden (Kleinschreibung). Andere Namen werden nicht gefunden.
- **Bildrechte:** Nur eigene Fotos oder Bilder mit gültiger Lizenz verwenden.
- **Dateigröße:** Für schnelle Ladezeiten vor dem Hochladen komprimieren (Ziel < 300 KB pro Bild).
- WebP wird von allen aktuellen Browsern unterstützt und ist meist kleiner als JPG — bei Nutzung
  die Endung in den `--img:url(...)`-Pfaden in `index.html` entsprechend auf `.webp` ändern.
- Über den Fotos liegt ein leichter Dunkel-Verlauf, damit Icons und Bildunterschriften lesbar
  bleiben.
